em (relativo ao elemento pai) A utilidade (em) é relativa ao tamanho da fonte do elemento pai Ou seja: se o elemento PAI tiver "font-size: 16px", entao 2em = 32px mas se o elemento nao for declarado, segue o jogo! Ainda sim, o navegador fará a conta com base nos 16px é mais previsivel e estavel, porque ignora o tamanho do elemento-pai Quando usar cada uma? em = ideal para (Margens Internas, espaçamentos adptaveis dentro de caixas/componentes) rem = ideal para tamanhos de font globais e layount consistentes
